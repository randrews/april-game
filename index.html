Garden Defense

A tower-defense game where you protect your flowers from bugs.

Download:

You can get the Löve file or a Windows build. The Löve version will work on any platform but requires you to install Löve2d.

How to play:

The goal of the game is simply to last as long as possible, and get as high a score as you can. You earn points by having flowers survive to maturity, but bugs will try to eat them.

Flowers maturing will give you money, which you can use to build and upgrade towers (to shoot bugs), till plots (to plant more flowers), and to spread insecticide (to slow down bugs).

Planting flowers requires seeds, but maturing flowers give you seeds. When you run out of seeds and flowers, you can no longer earn points, so the game ends.
